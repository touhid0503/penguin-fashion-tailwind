# 🐧 Penguin Fashion – Tailwind CSS Project

<img src="assets/icons/logo.png" width="220">

---

## 📌 Project Overview

**Penguin Fashion** is a modern and responsive **fashion e-commerce landing page** built using **Tailwind CSS**.
This project recreates the UI design provided in the blueprint and demonstrates how to build responsive layouts using Tailwind utility classes.

The website includes product sections, promotional banners, and service features commonly used in modern e-commerce websites.

---

## 🛠️ Technologies Used

* **HTML5** – Page structure
* **Tailwind CSS** – Styling and responsive layout
* **CSS3** – Additional custom styling

---

## 🎨 Project Features

### 1️⃣ Navigation Bar

The navigation bar contains:

* Brand logo
* Navigation links
* Shopping cart icon

The layout is responsive and adjusts properly across different screen sizes.

---

### 2️⃣ Hero Section

The hero section highlights the main promotional banner of the website.

It includes:

* Large promotional heading
* Product promotion text
* **Buy Now** button
* Model image

This section uses **flexbox and responsive Tailwind classes**.

---

### 3️⃣ Women Jacket Section

This section displays **three women's jacket products**.

Each product card contains:

* Product image
* Product title
* Product description
* Price
* **Buy Now button**

Cards are designed using **Tailwind grid layout**.

---

### 4️⃣ Men Jacket Section

Similar to the women's section, this section contains **three men's jacket products**.

Each card includes:

* Product image
* Product title
* Description
* Price
* Buy button

The cards are responsive and adapt to different screen sizes.

---

### 5️⃣ Service / Feature Section

This section highlights the services offered by the store.

Three service cards are included:

* **Find the Perfect Fit**
* **Free Exchanges**
* **Contact Our Seller**

Each card contains an icon and a short description.

---

## 📱 Responsive Design

The layout is fully responsive using Tailwind's responsive utilities.

| Screen Size | Layout Behavior                       |
| ----------- | ------------------------------------- |
| Mobile      | Sections stack vertically             |
| Tablet      | Product cards appear in two columns   |
| Desktop     | Product cards appear in three columns |

Tailwind responsive prefixes used:

* `sm:` – Small devices
* `md:` – Medium devices
* `lg:` – Large devices
